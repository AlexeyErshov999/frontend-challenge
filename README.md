# Cat Gallery

Веб-приложение для просмотра котиков с возможностью добавления в избранное. Данные загружаются динамически по мере прокрутки.

⚠️ **Важно:** Изображения котиков могут не загружаться без использования VPN — это связано с особенностями работы внешнего API. Если картинки не отображаются, включите VPN и обновите страницу.

## 🚀 Функциональные возможности

- ✅ **Главная страница** — по умолчанию открывается лента всех котиков
- ❤️ **Избранное** — добавление и удаление котиков в избранное (сохраняется в `localStorage`)
- 📜 **Infinite scroll** — динамическая подгрузка новых порций данных при прокрутке вниз
- 📱 **Адаптивный дизайн** — корректно отображается на всех устройствах: от мобильных телефонов до широких мониторов

## 🛠 Стек технологий

- **React**
- **TypeScript**
- **Redux Toolkit**
- **React Router DOM**
- **Axios**
- **Vite**
- **SCSS (Sass)**
- **classnames**

## 💡 Почему именно эти технологии?

Выбранный стек сочетает **высокую производительность** и **коммерческую востребованность**:

- **React + TypeScript** — стандарт современной frontend-разработки в большинстве компаний
- **Redux Toolkit** — наиболее распространённое решение для глобального состояния
- **Vite** — мгновенный Hot Module Replacement и быстрая сборка
- **SCSS + classnames** — удобная работа со сложной стилизацией, часто встречается в реальных проектах
- **Axios** — надёжный HTTP-клиент с удобными интерцепторами

Этот стек максимально привычен и позволяет быстро разрабатывать.
